# Centralised-Calling-System:
The Sign-In Page provides secure access with separate options for Admin Login and Client Login, ensuring users can access features tailored to their roles.
![image](https://github.com/user-attachments/assets/a9eab9dd-553c-44aa-bd51-3eed4afde235)

The Dashboard Page provides real-time visualizations powered by WebSocket connections, displaying metrics such as active calls per account and total active calls across different servers through interactive graphs and gauge charts. It ensures seamless monitoring of call activities for efficient decision-making.
![image](https://github.com/user-attachments/assets/12d634c8-c293-4f15-a97c-8e98182c32be)

The Create Client Page is for superusers to securely create and manage client profiles, ensuring centralized control.
![image](https://github.com/user-attachments/assets/d679d028-8022-4a93-a04b-1ce6badbefd7)

The Client Details Page provides a tabular view of all clients, including their account information, call statuses, and allocated servers, allowing superusers to manage and monitor client details efficiently.
![image](https://github.com/user-attachments/assets/217ee2e5-4f03-43c7-bdff-6a121d8e55e3)

The Client Details Table includes interactive icons for updating client information and resetting passwords. Clicking the Update Client icon opens a modal to modify client details, while the Reset Password icon triggers a modal for securely resetting the client's password.
![image](https://github.com/user-attachments/assets/4d498543-a7ed-4f7b-8fa9-a815d48d2bc1)  ![image](https://github.com/user-attachments/assets/823ca75f-700d-4ac0-b3cb-b7653a16479d)

The Manage Allocated Servers Tab allows users to add, update, or delete server details efficiently.
![image](https://github.com/user-attachments/assets/44c36aed-5743-4e58-b0a2-0d189e0997aa)

The Create Server Page enables superusers to add new servers with specific configurations.
![image](https://github.com/user-attachments/assets/dcc3a87c-1134-40a5-827a-7e76fbe899bc)

The Server Details Tab displays server information and includes an Update Modal for modifying server configurations.
![image](https://github.com/user-attachments/assets/1c456761-da7c-4aa1-9e75-cd47db4bd87e)

The Active Queues Page displays live data of all running queues, including their names, states, message statuses, and rates, ensuring real-time monitoring.
![image](https://github.com/user-attachments/assets/6a61bfae-e0ff-43d4-ac7f-424b356bebc2)

The Manage Server Tab under the Controls Page displays server details with their current status and provides options to change and save statuses. 
![image](https://github.com/user-attachments/assets/54f0d09f-95e1-4273-8b0e-6c4e1f2dad60)

The Manage Clients Tab under the Controls Page allows superusers to update and save the status of clients (e.g., active or inactive), ensuring effective client management.
![image](https://github.com/user-attachments/assets/585d1ada-e721-4a72-9734-0f148015385a)
![image](https://github.com/user-attachments/assets/1998842d-7133-4f1c-ac9b-8a12f89f9f9d)

The Call Queue Details Page combines insights from scheduled and quick calls, providing metrics such as answered, failed, busy, and unanswered calls for scheduled queues, along with real-time tracking of quick call statuses.
![image](https://github.com/user-attachments/assets/5f81842b-89bf-4e2c-af83-0b8510c6f1c9)
![image](https://github.com/user-attachments/assets/d71db63e-767b-4179-83ea-629ebc5aecda)

The Call Detail Records Page offers a detailed view of call logs with a Filter Modal to select specific servers and clients, it also includes a Download Button to export call records in CSV format for further analysis.
![image](https://github.com/user-attachments/assets/ffddec47-032f-4f9b-bcfc-5b6c79667609)

The Quick Call Page enables users to make instant calls by filling in required details such as customer information, server code, and endpoints.
